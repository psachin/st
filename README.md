## Usage

### Install

Install [lein](http://leiningen.org/#install)

### Setup

```bash
lein deps
```

### Test

```bash
lein run
```

### Compile

```bash
lein uberjar
```

### Run

```bash
java -jar target/st-0.1.0-SNAPSHOT-standalone.jar
```
